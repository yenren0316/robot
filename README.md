# 彈射機器人 (Ricochet Robots)

這是一個「彈射機器人 (Ricochet Robots)」遊戲專案。
彈射機器人是一款經典的解謎遊戲，玩家的目標是利用場地上的障礙物（牆壁）以及其他機器人，以最少的步數將特定顏色的「目標機器人」移動到指定位置。
請注意：機器人一旦開始移動，除非撞到牆壁或另一個機器人，否則不會停下來！

## 環境安裝

在執行遊戲前，請確保您的系統已安裝 Python 3，並安裝必要的套件 (Pygame)：

```bash
pip install pygame
```

## 如何遊玩

1. 開啟終端機 (Terminal) 或命令提示字元 (CMD)。
2. 進入專案所在的資料夾。
3. 輸入以下指令啟動遊戲：

```bash
python ricochet_pygame.py
```

### 操作方式說明
- **選擇機器人**: 透過滑鼠點擊來選取你要移動的機器人。
- **移動**: 使用 **鍵盤方向鍵 (上、下、左、右)** 控制機器人的移動方向。
- **提示**: 按下 **`H` 鍵** 可以開啟或關閉最佳解的圖示箭頭提示（直接畫在畫面上）。
- **重置**: 按下 **`R` 鍵** 可以將機器人退回原本的位置重新開始該局。
- **換目標**: 按下 **`空白鍵`** 可以在同一個地圖下尋找下一個目標。
- **重生地圖**: 按下 **`N` 鍵** 可以重新隨機生成一個全新的地圖與題目。
- **隱藏步數防雷**: 最佳解步數預設會被隱藏，點擊畫面任意處即可顯示。
- **規則**: 機器人會直線衝刺，直到撞上牆壁或其他機器人才會停下。請善用這個特性來轉彎，找到通往目標的終極路徑！
